BOOSE Application Documentation

Overview

Welcome to the documentation for the Medium Scale Software Engineering Project implementing the BOOSE (Basic Object Orientated Software Engineering) language with a graphical interface.

Features

  • Real-time command execution: Type commands and see immediate results
  • Graphical canvas: Visual drawing with pen position tracking
  • Program editing: Multi-line program support with syntax checking
  • File operations: Save/Load programs and images

Quick Navigation

  • API Documentation - View all classes and methods
  • Getting Started - Quick start guide

Project Structure

This application consists of:

  • Form1: Main application window and UI logic
  • AppCanvas: Drawing canvas implementation
  • AppCommandFactory: BOOSE command factory with custom commands
